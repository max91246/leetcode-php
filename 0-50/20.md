### :pencil2:题目分析
**最常规的解决就是定义一个三对括号相对应的键值对数组,定义一个格外的数组用来存储没有匹配的括号字符串。只要不匹配，就把他丢进数组中，每次取数组中最后一个元素和当前遍历的字符串位置值相比较，如果相等，说明匹配，从数组中拿走。最后判断数组是否为空即可。**
****
### :pencil2:解法一
```php
       /**
            * @param String $s
            * @return Boolean
            */
           function isValid($s) {
               $list=['{'=>'}','['=>']','('=>')'];
               $res=[];
               for($i=0;$i<strlen($s);$i++){
                   $tmp=end($res);
                   if($list["$tmp"] == $s[$i]){
                       unset($res[key($res)]);
                   }else{
                       $res[]=$s[$i];
                   }
               }
               return count($res)==0;
          }
```

### :pencil2:解法二
```php
 /**
     * @param String $s
     * @return Boolean
     */
    function isValid($s) {
       do{
            $s=str_replace(['()','[]','{}'],'',$s,$count);
        }while($count);
        return empty($s);
    }
```

function isValid($string){
    $status = 'true';
    $array = [
        '{' => '}',
        '[' => ']',
        '<' => '>',
    ];

    for ($i = 0; $i < strlen($string); $i++) {
        if($i%2===0 ){
          if ($array[substr($string,$i,1)] != substr($string,$i+1,1)){
              $status = 'false';
              break;
          }
        //   echo $array[substr($string,$i,1)] . substr($string,$i+1,1);
        }
    }   
    echo $status;
}
